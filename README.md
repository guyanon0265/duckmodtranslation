# Duck Mod Translation

[Duck Mod](https://modrinth.com/mod/mcduckmod)

I found it immersion-breaking for English words to appear in a non-English instance of Minecraft, and you might too.  
This repo exists to help coordinate the translation of Duck Mod. Minecraft supports over 130 languages, much too many to do by default.  
*To request a translation, create a new issue.  
To submit a translation, create a pull request with the JSON file.  
You can also DM me on Discord at 'axiomares' to do either.*
